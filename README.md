# Dynamic-List
A Java desktop application that simulates a cloud storage administrator system.  <br>
Users can add, remove, and search stored items through a graphical interface built using Java Swing.  <br>

## 🏗 Architecture (MVC Inspired)  <br>

- **Cloud.java (Model)**  <br>
  Handles all data storage and business logic.<br>

- **GUI.java (View + Controller)**  <br>
  Displays items and handles user interaction.<br>

- **Main.java**  <br>
  Entry point that launches the application.<br>

---

## ✨ Features<br>

✔ Add items  <br>
✔ Remove items  <br>
✔ Search items (returns index location)  <br>
✔ Dynamic list updates  <br>
✔ Object-Oriented Design  <br>
